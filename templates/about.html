About

Instant Dungeon is an RPG Dungeon Toolkit created for my Third-year project in conjunction with my advisor, Dr James Mapp. It has been designed and developed over the course of my third year at University and aims to deliver a rapid solution to creating RPG Dungeon Maps in a procedurally generated way. In addition, Instant Dungeon aims to offer a suitable method of populating these Dungeon Maps with appropriately levelled and themed enemies for a given Party size and Average Level.

How to use Instant Dungeon

Instant Dungeon has been designed in such a way that generating your next Dungeon Map is just a single click away. If you would like a completely random Dungeon based on our recommended parameters, simply click the 'Create' button on the home page, and you will be presented with a procedurally generated Dungeon Map populated with random enemies. If you would prefer to have some influence over the Dungeon's Layout and which enemies populate it, then fill in the creation parameters with your own settings and see your own Dungeon come to life.

Parameters Explained

  • Dungeon Seed: This parameter allows you to enter the Dungeon seed of a previously generated Dungeon and remake that exact same Map (This will only work if all of the parameters used are the same as the original). To generate a new Dungeon, leave this parameter at 0.
  • Dungeon Shape: This parameter allows you to select whether your Dungeon will be shaped like a rectangle or a square.
  • Dungeon Size: This parameter allows you to select the overall size of your Dungeon, ranging from Tiny (approx 25x20 squares) to Large (approx 70x50 squares).
  • Corridor Algorithm: This parameter allows you to select how the corridors connecting rooms in the Dungeon will be generated. Selecting 'BSP' will result in linearly connected corridors, with few turns, whereas selecting 'Drunkard's Walk' will result in winding corridors with many twists and turns.
  • Tileset: This parameter allows you to select which tileset is used to construct the Dungeon image.
  • Population Seed: Much like the Dungeon Seed parameter, this can be used to populate a previously generated Dungeon with the same enemies that filled it before (Again this will only work if all of the parameters used are the same as the original). To populate with random enemies, leave this parameter at 0.
  • Dungeon Theme: This parameter allows you to select which types of enemies will populate your Dungeon, ranging from Beasts to Undead (NOTE: the selection of some of these themes with specific party sizes and levels may result in no/few enemies being placed).
  • Enemy Density: This parameter allows you to decide what percentage of Dungeon Rooms will contain an encounter/enemies. This can range from scarce, 33%, to Full, 100%.
  • Party Size: This parameter allows you to populate your Dungeon with enemies appropriate for a specific amount of players, in conjunction with the Party Average Level parameter. Leave this at 0 for a random Party size between 3 and 5.
  • Party Average Level: This parameter allows you to populate you Dungeon with enemies appropriate for a specific Party level, in conjunction with the Party Size parameter. Leave this at 0 for a random Party Average level.
